# Headers

```
# This is a heading 1
## This is a heading 2
### This is a heading 3
#### This is a heading 4
##### This is a heading 5
###### This is a heading 6
```

You can also use the alternative syntax for Heading 1 and Heading 2.

```markdown
Heading 1
===

Heading 2
---
```
