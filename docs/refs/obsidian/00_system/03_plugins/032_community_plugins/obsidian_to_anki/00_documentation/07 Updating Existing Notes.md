# Updating Existing Notes

When the script successfully adds a note, it will append an ID to the Note Data. This allows you to *update existing notes by running the script again*.

Example output:

```
START
Basic
This is a test.
Back: Test successful!
<!--ID: 1566052191670-->;
END
```
